# Create Symlinks

Symlink `vimrc` to `~/.vimrc` and `vim` to `~/.vim`.

Symlink `tmux.conf` to `~/.tmux.conf`

Open vim and run `:PlugInstall`.

# Install Homebrew

`/bin/bash -c "$(curl -fsSL
https://raw.githubusercontent.com/Homebrew/install/master/install.sh)"`

# Install Tmux

`brew install tmux`

# Git Config

# Install Node

`brew install node`

# Install Yarn

`brew install yarn`

bash`
